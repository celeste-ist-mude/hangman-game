2. How does it work : Wins

I made a score system on my own, it is pretty simple but very effective.
The score are represented by buoys :
 › The red ones increase by one for each    victory
 › The blue ones increase by one for    each one hundred victory. It also    resets red buoys

3. How does it work : Game

It it very simple to play : You will see some squares (between one and seven) and your goal is to find the letters which should be in these boxes
Once you asked for a letter, either the letter is part of the word to find or the letter is not part and a coconut will start falling.
If you are wrong five times, the coconut will hit the human's head and he will die.
